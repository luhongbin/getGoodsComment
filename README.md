# getGoodsComment

商城的评论下载

## 代码用法

1. 首先进入官方PC网页 F12 获取链接信息和COOKIES 
2. 讲上述信息 填写到代码中
3. 由于代码执行速度快 而且分开了具体产品爬取 反爬没反应过来 数据到手
> 注意：如果出现爬取失败 重新获取COOKIES即可
